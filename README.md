\# Pi NAS Monitoring



A private Raspberry Pi NAS monitoring dashboard built with \*\*React (PWA)\*\* and \*\*ASP.NET Core\*\*.



The project provides a lightweight web interface for monitoring and managing a Raspberry Pi over a private Tailscale network.



\---



\## Features



\### Backend (.NET)



\- JWT authentication

\- Linux service monitoring

\- Storage monitoring by UUID

\- Configurable storage devices

\- REST API



\### Frontend (React)



\- Responsive admin dashboard

\- Progressive Web App (PWA)

\- Mobile friendly

\- Authentication

\- Service monitoring

\- Storage monitoring
